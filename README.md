# Sammlung der AP1-Zwischenprüfungen – Fachinformatiker

Willkommen in meiner Sammlung der **AP1 (Abschlussprüfung Teil 1)** Zwischenprüfungen für den Ausbildungsberuf **Fachinformatiker** (Anwendungsentwicklung / Systemintegration).

Diese Sammlung dient der Vorbereitung auf die Zwischenprüfung und soll eine zentrale Anlaufstelle für alte Prüfungen, Übungsaufgaben, Lösungen und Lernmaterialien sein.

## Inhalt

Die Sammlung enthält:
